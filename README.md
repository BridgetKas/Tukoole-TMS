# Tukoole
Tukoole ,a luganda word that means let's work,  is a talent managment system that provides talents to different at afforable prices. It is structured as a monorepo containing both the backend and frontend applications.

## Tech Stack

**Backend**
* Express (Node.js framework)
* TypeScript
* pnpm (Package manager)


**Frontend**
* Next.js (React framework)
* TypeScript
* Tailwind CSS
